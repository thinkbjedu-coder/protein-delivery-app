# 社内便 納品書作成システム

社内の複数事業所間で物品の送付・受領を管理するWebアプリケーションです。

## 機能

- 送付状・受領証の作成
- 送付記録の保存と履歴管理
- 受領確認と受取人名の記録
- 記録の削除機能
- 印刷機能

## 使用技術

- **フロントエンド**: HTML, CSS, JavaScript
- **バックエンド**: Node.js, Express
- **データベース**: JSONファイルベース

## ローカル環境での起動方法

```bash
npm install
npm start
```

ブラウザで `http://localhost:3000` にアクセスしてください。

## デプロイ

このアプリケーションはRenderにデプロイ可能です。
